# 🚀 GCC Creative Studio has moved!

This project has moved to a new repository:
👉 **[https://github.com/GoogleCloudPlatform/gcc-creative-studio](https://github.com/GoogleCloudPlatform/gcc-creative-studio)**

## What this means for you

If you have already deployed Creative Studio from this repository, please follow these steps to migrate to the new repository:

1. **No Data Loss:** You will **not** lose any data. Your PostgreSQL database and other resources will remain intact.
2. **Redeploy Required:** You will need to run Terraform again and redeploy Creative Studio.
3. **Follow New Instructions:** Please follow the video and steps detailed in the new repository to redeploy:
   👉 [Deploy in 20 Minutes](https://github.com/GoogleCloudPlatform/gcc-creative-studio?tab=readme-ov-file#deploy-in-20min)
4. **New Fork:** You will need to fork the new repository and update your configuration (like Cloud Build triggers) to point to your new fork by running Terraform.

After completing these steps, you will be ready to use the platform from the new source!
